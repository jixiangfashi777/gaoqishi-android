## 第1步：启用GitHub Pages（2分钟）

### 📝 操作步骤

1. **打开浏览器，访问仓库设置页面**
   ```
   https://github.com/jixiangfashi777/gaoqishi-android/settings/pages
   ```

2. **配置GitHub Pages**
   - 在"Build and deployment"区域
   - **Source**: 选择 `Deploy from a branch`
   - **Branch**: 选择 `main` 分支
   - **Folder**: 选择 `/ (root)`
   - 点击 **Save** 按钮

3. **等待部署完成**
   - 页面会刷新并显示：
     ```
     ✅ Your site is published at https://jixiangfashi777.github.io/gaoqishi-android/
     ```
   - 大约需要1-2分钟

4. **验证部署**
   - 点击上面的URL或直接访问：
     ```
     https://jixiangfashi777.github.io/gaoqishi-android/
     ```
   - 确认页面正常显示
   - 测试打开一篇文章，看TTS按钮是否出现

## 第2步：使用PWA Builder生成APK（10分钟）

### 📝 操作步骤

1. **访问PWA Builder**
   - 打开浏览器，访问：
     ```
     https://www.pwabuilder.com/
     ```

2. **输入你的网站URL**
   - 在首页大大的输入框中输入：
     ```
     https://jixiangfashi777.github.io/gaoqishi-android/
     ```
   - 点击紫色的 **"Start"** 按钮

3. **等待分析完成**
   - PWA Builder会自动分析你的网站
   - 显示PWA得分（70-100分都正常）
   - 分析时间约10-30秒

4. **选择Android平台**
   - 在"Platform Options"区域找到 **"Android"** 标签
   - 点击 **"Store Package"** 按钮

5. **配置应用信息**
   填写以下内容：
   
   ```
   App Name: 高七师佛学心理学
   Short Name: 高七师
   Package ID: com.gaoqishi.app
   App Version: 1.0.0
   Version Code: 1
   Theme Color: #C41E3A
   Background Color: #FFFFFF
   Display Mode: standalone
   Orientation: portrait
   ```
   
   **Signing Options（签名选项）**：
   - 选择 ☑️ **"Generate new signing key"**（生成新密钥）
   
   **重要**：如果你计划将来更新APP，请保存生成的密钥文件！

6. **生成APK**
   - 选择包类型：**TWA (Trusted Web Activity)** 推荐
   - 点击 **"Download"** 按钮
   - 等待下载（文件约2-5MB）

7. **解压ZIP文件**
   - 下载的文件名类似：`gaoqishi-android-signed.zip`
   - 解压后找到：`app-release-signed.apk`
   - 这就是你的Android安装包！

---

## 第3步：安装到手机测试（5分钟）

### 📝 操作步骤

1. **准备Android手机**
   
   **启用"未知来源"安装**：
   - 设置 → 安全
   - 找到"未知来源"或"安装未知应用"
   - 启用你用来传输APK的应用（如文件管理器、微信、浏览器）

2. **传输APK到手机**
   
   选择一种方式：
   
   **方式A：USB传输（最快）**
   ```bash
   adb install app-release-signed.apk
   ```
   
   **方式B：微信传输（最方便）**
   - 电脑上打开微信，发送APK文件给"文件传输助手"
   - 手机上打开微信，下载APK文件
   
   **方式C：云盘传输**
   - 上传到百度网盘/Google Drive
   - 手机上下载
   
   **方式D：邮件传输**
   - 发送到自己的邮箱
   - 手机上下载附件

3. **安装APK**
   - 在手机上点击APK文件
   - 点击"安装"
   - 等待安装完成（5-10秒）
   - 点击"打开"启动应用

4. **测试功能**
   
   **基本功能测试**：
   - [ ] 应用成功启动
   - [ ] 首页文章列表加载正常
   - [ ] 底部导航栏工作正常
   - [ ] 各页面切换流畅
   
   **TTS功能测试**：
   - [ ] 点击任意文章打开详情页
   - [ ] 右下角出现红色圆形播放按钮
   - [ ] 点击播放按钮
   - [ ] 听到中文语音朗读文章
   - [ ] 按钮有脉冲动画
   - [ ] 点击暂停按钮，朗读停止
   - [ ] 返回首页，按钮消失
   
   **其他功能测试**：
   - [ ] 分类页面正常
   - [ ] 资源下载功能正常
   - [ ] 网络加载正常

## ❓ 常见问题快速解答

### Q1: GitHub Pages显示404错误
**A**: 等待1-2分钟让部署生效，然后刷新页面。

### Q2: PWA Builder分析失败
**A**: 确认GitHub Pages已启用且URL可访问，URL后面不要加多余的斜杠。

### Q3: APK安装失败
**A**: 检查是否已启用"未知来源"安装，确保手机存储空间充足（至少50MB）。

### Q4: TTS无声音
**A**: 检查手机音量，确认设备已安装中文语音包（设置 → 语言与输入 → 文字转语音输出）。

### Q5: 文章加载失败
**A**: 检查手机网络连接，确认原网站API正常（https://gaoqishi.pages.dev）。
